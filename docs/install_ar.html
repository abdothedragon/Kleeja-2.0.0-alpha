# : طريقة التركيب

1 - قم برفع النسخه إلى الخادم بأي برنامج FTP


2- 3- شغل معالج التثبيت عن طريقة الرابط
http://sitename.com/kleeja/install/
وبدل :
sitename.com : دومين موقعك
kleeja : مجلد سكربت كليجا

4- احذف ملف التثبيت السابق .. للأمان
5- إدخل موقعك الآن وستجد كل شيء بشكل جيد .. للدخول للوحة التحكم أدخل كعضو وسوف تجد رابط اسفل السكربت للمسؤول
6- لا تحاول نزع الحقوق من السكربت , فنزع الحقوق هو نكران للجميل .. إذا أردت إزالة الحقوق راسلنا لشراء حقوق النسخه بثمن بسيط جد لكي نستمر بالتطوير ..

# : طريقة الربط بالمنتديات او بالسكربتات

1- بعد تثبيت السكربت كما هو موضح بالنقطه السابقه .
2- قم بفتح لوحة التحكم والذهاب "للإعدادات".
3- قم بتعديل خيار "نظام العضويات" لنوع المنتدى المراد .
4- قم بتعديل ملف includes/config.php الجزء الثاني منه
اضف كالتالي [كمثال]
//for integration with other scripts [ must change user systen from admin cp ]
$script_path = './vb';// مجلد السكربت كمثال vb
$script_encoding = 'windows-1256';// ترميز الصفحات السكربت

او الربط من قاعدة البيانات مباشرة كتالي
[كمثال]
//for integration with other scripts [ must change user systen from admin cp ]
$script_encoding = 'windows-1256';// ترميز الصفحات السكربت
$script_srv = 'localhost';
$script_db = 'vb';
$script_user = 'root';
$script_pass = 'pass';

   لايمكنك استخدام الطريقه الثانية phbb3 ملاحظة : اذا كنت تسخدم منتدى
احفظ الملف واذهب للدخول , اذا كنت مديرا بالسكربت سوف يتعرف عليك كمدير للسكربت ..

# : كيف أقوم بجعل الروابط هتمل للتتوافق مع SEO

1- من لوحة التحكم إذهب لـ "اعدادات"
2- قم بتفعيل خيار "روابط هتمل"
3- قم بنقل ملف docs/.htaccess.txt إلى المجلد الرئيسي
4- بعد نقله قم بتغيير الاسم إلى .htaccess
استمتع بالروابط ..
